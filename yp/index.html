Privacy Policy

The "Year Progression" bot prioritises user privacy by refraining from storing any user data.
The Channel ID of the subscribed channel gets sent to a secure Discord Server dedicated to tracking the bot's performance.
This server logs successful updates and errors, and support in user issues.
The subscribed channel id is also stored in our database.

If any user has issues with our bot, our support team requests the Channel ID and the reviews logs, and provides assistance.
It's important to note that we have no access to any user data beyond the Channel ID, which has limited usability.
Rest assured, we do not sell any data.

Please be aware that @SerStars, the developer, reserves the right to modify this policy at any time and without notice.
